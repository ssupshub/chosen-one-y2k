# YOU HAVE BEEN CHOSEN
## A Retro Y2K Multi-Page Hero Website

A fictional early-2000s style website built with plain HTML, CSS, and JavaScript.
No frameworks. No build tools. Open index.html and go.

---

## FILE STRUCTURE

```
chosen_world_site/
├── index.html        Home page — visitor counter, world status, mission intro
├── mission.html      Interactive mission checklist with XP tracking
├── prophecy.html     The prophecy scrolls with a working oracle
├── enemies.html      Enemy database with battle and filter system
├── weapons.html      Armory with drag-to-equip loadout builder
├── alliances.html    Ally roster with live messaging and chat
├── world.html        ASCII world map with clickable regions
├── guestbook.html    Paginated guestbook with a working sign form
├── contact.html      Contact form with subject and threat level
├── style.css         Shared styles used by all pages
└── README.md         This file
```

---

## HOW TO RUN

Double-click index.html in any modern browser.
No server required. No installation. No dependencies.

Works in: Chrome, Firefox, Edge, Safari.

---

## FEATURES

- 9 fully linked pages with shared navigation
- Animated scanline overlay and CRT flicker effect
- Live visitor counter and mission countdown timer
- Interactive mission checklist with XP tracking
- Enemy database with search, filter, and battle system
- Weapon armory with equippable loadout slots
- Ally messaging with simulated responses
- Clickable ASCII world map with region detail panel
- Paginated guestbook with live entry submission
- Oracle question system on the prophecy page
- Contact form with validation and send confirmation

---

## NOTES

All data is in-memory only. Entries reset on page reload.
Fonts load from Google Fonts. An internet connection is needed for correct typography.
If offline, the site falls back to Courier New.
